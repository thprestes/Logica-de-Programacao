**Mini-Projeto: Caverna do Dragão**
Este código foi desenvolvido em **VisualG** para exercitar estruturas de repetição e seleção.

Conceitos usados: Variáveis (Inteiro, Logico), Estrutura de repetição (Enquanto), Estrutura Condicional (Escolha/Caso).

Objetivo: Criar um mini jogo de escolhas onde o jogador deve encontrar a saída da caverna.

**Como funciona:**
1. O jogador escolhe entre 3 portas.
2. O sistema verifica a escolha usando 'ESCOLHA/CASO'.
3. O jogo reinicia automaticamente usando 'ENQUANTO' até que o jogador vença.
